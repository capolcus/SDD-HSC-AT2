Ashu's Anatomy App

Description:
Ashu's Anatomy App is a GUI-based educational application for learning about the human skeletal and muscular systems. Users can view detailed information about different bones and muscles, take quizzes to test their knowledge, and track their progress.

Prerequisites:
1. Python 3.6+
2. pip
3. Tkinter
4. customtkinter

Installation Instructions:
1. Ensure you have Python installed. Download from https://www.python.org/downloads/ if needed.
2. Open a terminal or command prompt.
3. Install customtkinter by running the following command:
   pip install customtkinter

Directory Structure:
Ensure your project directory has the following structure:

anatomy-app/
│
├── skeletal_system.png
├── muscular_system.png
├── main.py
├── README.txt (this file)

Running the Application:
1. Open a terminal or command prompt.
2. Navigate to the project directory:
   cd path/to/anatomy-app
3. Run the application:
   python main.py

Usage:
1. Main Menu:
   - Learn Anatomy: Choose to learn about either the skeletal or muscular system. Select a part to view detailed information.
   - Quizzes: Select a quiz (skeletal or muscular system) to test your knowledge.
   - Progress: View your quiz progress and scores.
   - Settings: Change the theme and text size of the application.
   - Exit: Close the application.

2. Learning Anatomy:
   - Choose either the Skeletal or Muscular system.
   - Select a part from the dropdown menu to view detailed information.

3. Taking Quizzes:
   - Select the quiz category (Skeletal or Muscular).
   - Answer the questions presented one by one.
   - View your score at the end of the quiz.

4. Settings:
   - Choose a theme from the dropdown menu.
   - Select a text size.
   - Apply the changes.
